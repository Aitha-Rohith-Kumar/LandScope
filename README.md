# 🏡 LandScope – AI Driven Land Investment Analyzer

## 📌 Overview
LandScope is an AI-powered platform that helps users analyze and predict land investment opportunities using machine learning.

## 🚀 Features
- 📊 Investment Score Calculation
- 🤖 ML Models (Linear Regression, Random Forest, K-Means)
- 🗺️ Interactive Map Visualization (Folium)
- 📉 Future Price Prediction
- 📄 PDF Report Generation
- 💰 Budget-Based Filtering

## 🛠️ Tech Stack
- Python (Django)
- Machine Learning (Scikit-learn)
- Pandas, NumPy
- Folium (Map Visualization)
- HTML, CSS, JavaScript

## 📂 Project Structure
- data_pipeline → Data processing
- core → Main Django app
- ml_engine.py → ML logic
- import_csv.py → Data loading
